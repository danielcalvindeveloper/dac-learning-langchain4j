# DAC Learning LangChain

Curso práctico de LangChain4j para desarrolladores Java.

## ¿Qué es DAC Learning LangChain?

DAC Learning LangChain es un recorrido práctico y progresivo para aprender a construir aplicaciones basadas en Inteligencia Artificial utilizando Java.

A diferencia de la mayoría de los cursos disponibles, que se enfocan principalmente en Python, este proyecto busca enseñar los mismos conceptos desde la perspectiva de un desarrollador Java empresarial.

## Objetivos

- Comprender los fundamentos de los LLM.
- Utilizar LangChain4j desde Java.
- Construir asistentes conversacionales.
- Implementar memoria conversacional.
- Diseñar Prompt Templates reutilizables.
- Obtener resultados estructurados mediante POJOs.
- Implementar soluciones RAG.
- Integrar herramientas externas.
- Construir agentes autónomos.
- Orquestar flujos complejos mediante LangGraph4j.
- Comprender MCP.
